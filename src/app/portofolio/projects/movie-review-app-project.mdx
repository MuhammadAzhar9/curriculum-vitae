## Overview

Didalam projek ini membuat sebuah website mengenai review film. Dapat melihat movie, memberikan rating kepada movie, memasukan film kedalam list tontonan dan memberikan suka pada film.

## Key Features

#### Client-Side Features
1. **Home Page**  
   The main landing page for users, providing access to movie information.
2. **User Authentication**  
   - **Register**: Allows new users to create an account.  
   - **Login**: Enables existing users to log in.  
   - **Forgot Password**: Provides a method for users to reset forgotten passwords.  
   - **Reset Password**: Enables users to reset their password using a token.  
3. **Movie Details** 
   - **Detail Movie Page**: Displays detailed information about specific movies, including reviews, actor information, and trailers.  
     - **Movie Trailer**  
     - **User Reviews**  
     - **Actor Information**: Includes details about actors and their roles in the movie.  
4. **Review Bar**  
   Users can add reviews and ratings for movies.

#### Admin-Side Features
1. **Dashboard**  
   Provides an overview of platform statistics and activities.
2. **Movie Management**  
   - **Input Drama**: Allows admins to add new movies.  
   - **List Drama**: Displays a list of all movies.  
   - **Validate Drama**: Enables validation and approval of movies.  
   - **Validate History**: Displays the history of validated movies.  
   - **Movie Trash**: Manages movies that have been moved to the trash.  
3. **Review Management**  
   Admins can manage user reviews.
4. **Actor Management**  
   Admins can manage actors and their roles in movies.
5. **Genre Management**  
   Admins can manage movie genres.
6. **Country Management**  
   Admins can manage country-related movie data.
7. **Awards Management**  
   Admins can manage movie awards.
8. **User Settings**  
   Allows admins to manage user settings and permissions.

## Technologies Used

* **React**: Untuk membangun **user interface (UI)** yang interaktif dan efisien dengan komponen yang bisa digunakan kembali.
* **React Router**: Untuk mengatur **routing** atau navigasi antar halaman dalam aplikasi *single-page* React.
* **Sass**: Untuk menulis **CSS** dengan fitur tambahan seperti variabel, *nesting*, dan *mixins* agar *stylesheet* lebih terorganisir dan efisien.
* **React-Bootstrap**: Untuk mengembangkan **UI** lebih cepat dengan komponen Bootstrap yang sudah diimplementasikan dalam React.
* **Material UI**: Untuk menyediakan **komponen UI** yang siap pakai dan mengikuti pedoman Google Material Design, mempermudah pembuatan antarmuka yang konsisten dan menarik.
* **Google OAuth**: Untuk **mengamankan autentikasi pengguna** dengan memungkinkan *user* masuk menggunakan akun Google mereka.
* **JWT (JSON Web Tokens)**: Untuk membuat **token akses** yang aman dan *stateless* untuk otorisasi, memungkinkan server memverifikasi identitas pengguna tanpa harus menyimpan sesi.
* **Passport.js**: Untuk mempermudah implementasi **berbagai strategi autentikasi** (seperti OAuth, JWT, *username/password*) di Node.js.
* **Express**: Untuk membangun **backend (API)** yang cepat dan fleksibel di Node.js, menangani permintaan HTTP dan *routing*.
* **MySQL**: Untuk mengelola **database relasional**, menyimpan, dan mengambil data aplikasi dengan efisien.
